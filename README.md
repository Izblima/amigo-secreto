# Amigo Secreto 🎁🎉

Um sistema simples para gerenciar sorteios de amigos secretos, permitindo adicionar nomes, sortear e exibir resultados.

---

## Funcionalidades ✨

- **Adicionar amigos**: Insira os nomes dos participantes para o sorteio.
- **Sortear**: Realize o sorteio de forma aleatória e exiba os pares de amigos secretos.
- **Interface amigável**: Design simples e intuitivo para facilitar o uso.

---

## Tecnologias Utilizadas 🛠️

- **HTML**: Estrutura da página.
- **CSS**: Estilização e design responsivo.
- **JavaScript**: Lógica de sorteio e interatividade.
