 Amigo Secreto 

Um sistema para sortear amigos secretos. Adicione os nomes dos participantes e faça o sorteio!

---

 Como Usar 

1. Adicionar amigos:
   - Digite o nome de um amigo no campo de texto e clique em "Adicionar".
   - O nome será exibido na lista abaixo.

2. Sortear:
   - Após adicionar pelo menos 2 nomes, clique em "Sortear Amigo".
   - Os pares de amigos secretos serão exibidos na seção de resultados.
